# APG Korea 업무보고 입력 가이드

## 📋 일일 업무보고

### 기본 형식
```
/보고 [업무내용] [시간]
```

### 예시
```
/보고 EL AL 요금표 정리 2h
/보고 GSSA 제안서 초안 작성 3h
/보고 인천공항 라운지 점검 1.5h
```

### 한 번에 여러 건
```
/보고
EL AL 요금표 정리 2h
GSSA 제안서 초안 3h
항공사 미팅 1.5h
```

### 상태 표시 (생략하면 자동 판단)
- ✅ 완료: `/보고 EL AL 요금표 정리 완료 2h`
- 🔄 진행중: `/보고 GSSA 제안서 작성중 3h`
- 📅 내일예정: `/보고 항공사 미팅 예정 1.5h`

### 그냥 말해도 됨
```
오늘 EL AL 미팅 2시간 했고, 제안서 초안 3시간 썼어
```
→ 김대리가 알아서 파싱합니다

## 🚗 외근 등록

```
/외근 [날짜] [장소] [목적] [시간]
```

### 예시
```
/외근 4/10 강남 EL AL 사무실 GSSA 계약 미팅 10:00-12:00
/외근 내일 인천공항 T2 라운지 점검 08:00-14:00
/외근 4/15 웨스틴조선 EL AL 미팅 14:00-16:00
```

### 그냥 말해도 됨
```
내일 강남 EL AL 사무실 외근 가요, GSSA 계약 협의
```

## 🏖️ 휴가 신청

```
/휴가 [날짜] [유형] [사유]
```

### 휴가 유형
- **연차**: 1일
- **반차(오전)** / **반차(오후)**: 0.5일
- **특휴**: 경조사 등
- **병가**: 질병

### 예시
```
/휴가 4/14 연차 개인사정
/휴가 4/15-4/16 연차 가족행사
/휴가 내일 반차(오후) 병원
/휴가 4/20 특휴 조부모 생신
```

### 그냥 말해도 됨
```
4월 14일 연차 쓸게요, 개인사정입니다
```

## ❗ 이슈 / 리스크

```
/이슈 GSSA 계약 법무 검토가 필요합니다
/이슈 EL AL 요금 수정 건, 응대 지연 중
```

---

## 💡 팁

- **시간 단위**: `2h`, `1.5h`, `30m` 모두 가능
- **날짜**: `4/10`, `내일`, `모레` 모두 인식
- **명령어 안 써도 됨**: 그냥 대화하듯 적으면 김대리가 알아서 분류
- **수정**: 다시 보내면 최신 내용으로 덮어씀
- **질문**: 뭘 쓰면 되는지 모르겠으면 그냥 `"오늘 뭐했어?"` 라고 물어보세요
